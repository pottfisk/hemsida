This site is self-hosted on a Raspberry Pi 3b+ running headless Raspbian Stretch.
The server uses Apache2 server software. DNS is handled by Loopia. SSL certificate is from Let's Encrypt to provide HTTPS.
This site is only a hobby creation to learn more about HTML, CSS, JS, Apache2 etc.
The page uses cookies to store preference of dark mode, the cookies are cleared when the browser is closed or after 2 days.
